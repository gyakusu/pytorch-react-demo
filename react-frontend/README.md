# 機械学習予測システム - Reactフロントエンド

このプロジェクトは、Fast APIバックエンドと連携するReactフロントエンドアプリケーションです。

## 機能

- 5つの特徴量を入力
- 機械学習モデルによる予測の実行
- 予測結果と信頼度の表示
- レスポンシブデザイン

## セットアップ

### 1. 依存関係のインストール

```bash
cd react-frontend
npm install
```

### 2. アプリケーションの実行

#### バックエンド（Fast API）の起動
```bash
# ルートディレクトリで
python api.py
```

#### フロントエンド（React）の起動
```bash
cd react-frontend
npm start
```

アプリケーションは http://localhost:3000 で利用できます。
APIサーバーは http://127.0.0.1:8100 で動作します。

## 使い方

1. Webブラウザで http://localhost:3000 にアクセス
2. 5つの特徴量フィールドに数値を入力
3. 「予測実行」ボタンをクリック
4. 予測結果（クラスと信頼度）が表示されます

## 機能詳細

- **入力検証**: すべてのフィールドが数値であることを確認
- **エラーハンドリング**: API通信エラーや入力エラーを適切に表示
- **ローディング状態**: 予測実行中の状態を表示
- **レスポンシブデザイン**: モバイルデバイスに対応
- **信頼度表示**: 予測の信頼度をパーセンテージとプログレスバーで表示
